# Suma_n_Numeros
Suma de los n primeros números naturales

# Analisis 

### Input 

- **N**: El número de términos a sumar (en este caso,9).

### Processing

- **S**: Suma de los primeros n naturales 

s= (n*(n+1))/2

### Output

- **S**: El resultado de la suma de los primeros N números
